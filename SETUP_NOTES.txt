BarterLab environment files

Files:
- requirements.txt : pip-style dependency list
- environment.yml  : conda environment definition

Recommended setup:

1) Create the conda env
   conda env create -f environment.yml

2) Activate it
   conda activate barter

3) Register the notebook kernel (optional)
   python -m ipykernel install --user --name barter --display-name "barter"

Notes:
- Python 3.11 is chosen for broad cross-platform compatibility.
- API keys should be stored in a .env file or environment variables, not directly in config files.
